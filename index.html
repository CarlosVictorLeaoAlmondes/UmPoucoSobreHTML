Introdução

Primeiramente, o que é HTML? HTML, que é a sigla para HyperText Markup Language, é uma linguagem de marcação utilizada para o desenvolvimento de sites, através dessa tecnologia e outras como por exemplo, CSS e JavaScript, é possível navegar pela internet de forma cada vez mais prática e agradável.

História

Essa linguagem de marcação surgiu em 1991, foi criada por Tim Berners-Lee, no CERN (European Council for Nuclear Research), na Suíça. Inicialmente, foi desenvolvido com uma ferramenta que permitia acrescentar alguns elementos nos textos usados por ele e um grupo de pesquisadores, era possível por exemplo, destacar certas partes do texto, vinculando com outro artigo que já estivesse publicado e que o primeiro artigo fizesse referência a ele. Esses foram os primeiros passos da linguagem, mas ela já evoluiu bastante, tornando possível a criação de sites como os utilizados atualmente.

Estrutura

Um arquivo html consiste de diversas tags, umas tags com fechamento e outras sem. Elas servem para “dizer” ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser título, parágrafo, imagem, link, blocos específicos da página, como por exemplo, head, header, body, footer e etc.

<p>Exemplo de parágrafo</p>

Neste exemplo foi utilizada a tag <p>, que serve para criar parágrafos. Essa é uma tag com fechamento, onde a abertura é <p> e o fechamento é </p>.

<img src="exemplo-de-imagem.png" alt="Breve descrição da imagem">

Agora foi utilizado o exemplo de uma tag sem fechamento, onde tem a abertura, mas não tem o fechamento. A tag utilizada foi a <img>, que serve para colocar imagens na página que está sendo criada.
